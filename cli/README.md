# Usage
<!-- usage -->
```sh-session
$ npm install -g valist-cli
$ valist COMMAND
running command...
$ valist (-v|--version|version)
valist-cli/0.0.4 darwin-x64 node-v14.17.2
$ valist --help [COMMAND]
USAGE
  $ valist COMMAND
...
```
